Keywords & Examples

SupLang maps everyday slang to JavaScript concepts. Here's everything you need to speak the language fluently.

Keywords

flex → console.log I / O

Print any value to the output. Show the world what you've got.

flex "Hello, World!"
flex 42
flex myVar
vibeCheck → if Control

Conditional branching. Only runs if the vibe is right (truthy).

vibeCheck (score > 50) {
  flex "You passed!"
}
lowkey → let Variable

Declare a mutable variable. Can be reassigned later — keeping it lowkey for now.

lowkey score = 0
lowkey name = "Mohit"
score = 100 // reassignment OK
bet → const Variable

Declare a constant. Once you bet, there's no going back — immutable.

bet PI = 3.14159
bet APP_NAME = "SupLang"
// PI = 4 → error, no cap
